## ✨ Features

- 🎨 Interactive buttons for offerings like **Laddu**, **Barfi**, **Fruit**, and **Flower**.
- 🌟 Particle animations triggered by button clicks.
- 📱 Responsive and visually appealing design.
- 💡 Custom glowing effects for buttons.
